Please input the following command on linux terminal for Running Reliable UDP Program (sender and receiver as well):

1). How to Compile and Run Receiver:
	Compiling:
 		gcc -o receiver receiver_RUDP.c
	Running:
		./receiver <port# in integer>
	Example:
		./receiver 55000

2). How to Compile and run sender:
	Compiling:
		gcc -o sender sender_RUDP.c
	Running:
		./sender <server IP Address> <server Port#> <filename.extenion>
	Example:
		./sender 127.0.0.1 55000 sample1.mp4
